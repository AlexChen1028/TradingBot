### C. 訊號條件落地（可程式化規則）

下面這些是直接可以變成 `if-else` 的條件：

```python
# 嘎空燃料偵測（向上）
SQUEEZE_FUEL_UP = (
    close_price > ema200 * 0.98 and          # 接近 200 日均線
    funding_rate < -0.0001 and               # 資費為負
    rsi < 70                                 # 還沒過熱
)

# 假突破風險（向下洗盤）
FAKE_BREAKOUT_RISK = (
    funding_rate > 0.0005 and                # 多頭過熱
    abs(close - resistance) / close < 0.005  # 緊貼壓力
)

# 短期持有者拋壓觸發
STH_CAPITULATION = (
    close_price < short_term_holder_cost_basis and  # 跌破成本線
    funding_rate > 0                                 # 多頭尚未投降
)

# 右側交易確認（高確定性進場）
RIGHT_SIDE_LONG_ENTRY = (
    close_price > ema200 and                 # 站上牛熊分界
    consecutive_closes_above_ema200 >= 3 and # 連續站穩
    regime == 'trending'                     # 趨勢市
)
```

### D. 風險警示（KOL 強調但 bot 已部分覆蓋）

- 不要在支撐位下方搶空 → bot 的 THRESHOLD=0.50 已避免在中性區進場，但可加入「靠近強支撐時不開空」的條件
- 多頭高度集中時的洗盤風險 → 可在 `open_position` 加入「資費 > 0.0005 時暫不重倉做多」的緩衝
- 法案投票/重大事件日的避險拋售 → 可加入 macro event blackout window，事件前 24h 不開新倉

## 四、實作技巧與可程式化規則

- **兩手準備規則（IF/ELIF）**：
  ```
  IF 多頭贏 (站穩壓力位): THEN 找新高位壓制做空
  ELIF 跌破支撐確認: THEN 等回測再追空
  ELSE: 不進場，等明確訊號
  ```
- **嚴禁追漲殺跌**：行情已到支撐位（地板）時絕對不追空 → `IF (price - support) / support < threshold AND signal == 'SHORT': SKIP`
- **週末交易法則**：`IF is_weekend AND no_preset_entry: SKIP`，給點位才做，沒給就擺爛。
- **區間進場技巧**：壓力位往高處抓一點，給予區間（如 79,200 ~ 79,700）以應對小幅突破。
- **成本保護機制**：達到預期支撐位/獲利目標時，**先出部分倉位**保護利潤，剩餘倉位止損上移至開倉價（保本機制）。
- **遇支撐獲利了結**：下跌觸及強支撐時果斷平倉空單或至少出一半，等明確跌破才再進場。

- **山寨幣「補漲」陷阱**：BTC 高位震盪時，散戶易產生「其他幣補漲」預期，把資金轉向 ETH/SOL/山寨。但這往往是主力陷阱，大陰線砸盤後幾天內跌破平台。〔5.18〕
- **整體市場信心轉向弱勢**：上漲趨勢結束，重回熊市中期階段，警惕「跌得多就抄底」心態。〔5.18〕

## 四、實作技巧與可程式化規則

**核心 IF/THEN 規則**：

```python
# 1. 反抽開空（破位交易）
IF prev_close < support_79000 AND current_price reaches 78500..79000:
    OPEN_SHORT()   # 反抽未站回支撐 → 順勢開空

# 2. 中軌支阻互換高空
IF kline_4h_close < bollinger_mid AND price rebounds to bollinger_mid:
    OPEN_SHORT()   # 中軌支阻互換，目標下軌

# 3. 強支撐摸底做多（謹慎、快進快出）
IF current_price in [75500, 76000]:
    OPEN_LONG_SMALL()  # 目標利潤僅 500-1000 點，遇 79000 強制平倉

# 4. 重倉禁區
IF current_price in [80000, 81500]:
    DO_NOT_OPEN_LONG()  # 中軌壓力區，絕對不接多

# 5. 120 天週期警示
IF days_since_last_swing_high >= 115:
    risk_level += 1
    max_pos_pct *= 0.5  # 自動降倉，準備變盤

# 6. 試錯單管理
IF entry_filled AND opposite_signal_kline (e.g. big red K against long):
    REDUCE_POSITION()   # 立即減倉，不被動持倉

# 7. 山寨幣減倉
IF market_regime == 'bearish_breakdown':
    SELL_ALTCOIN_SPOT()  # 流動性枯竭階段全面減倉山寨
```

**風險控制原則**：
- 高位空單跌破 77,000 開始陸續止盈，出現收針 → 完全平倉
- 小倉位試錯 → 確認後加碼 → 反向大 K → 果斷減倉
- 做多時利潤目標放小（500-1000 點），遇壓力勤快止盈
- 禁止盲目攤平 / 被動持倉
