### C. 訊號條件落地（可程式化規則）

下面這些是直接可以變成 `if-else` 的條件：

```python
# 嘎空燃料偵測（向上）
SQUEEZE_FUEL_UP = (
    close_price > ema200 * 0.98 and          # 接近 200 日均線
    funding_rate < -0.0001 and               # 資費為負
    rsi < 70                                 # 還沒過熱
)

# 假突破風險（向下洗盤）
FAKE_BREAKOUT_RISK = (
    funding_rate > 0.0005 and                # 多頭過熱
    abs(close - resistance) / close < 0.005  # 緊貼壓力
)

# 短期持有者拋壓觸發
STH_CAPITULATION = (
    close_price < short_term_holder_cost_basis and  # 跌破成本線
    funding_rate > 0                                 # 多頭尚未投降
)

# 右側交易確認（高確定性進場）
RIGHT_SIDE_LONG_ENTRY = (
    close_price > ema200 and                 # 站上牛熊分界
    consecutive_closes_above_ema200 >= 3 and # 連續站穩
    regime == 'trending'                     # 趨勢市
)
```

### D. 風險警示（KOL 強調但 bot 已部分覆蓋）

- 不要在支撐位下方搶空 → bot 的 THRESHOLD=0.50 已避免在中性區進場，但可加入「靠近強支撐時不開空」的條件
- 多頭高度集中時的洗盤風險 → 可在 `open_position` 加入「資費 > 0.0005 時暫不重倉做多」的緩衝
- 法案投票/重大事件日的避險拋售 → 可加入 macro event blackout window，事件前 24h 不開新倉

## 四、實作技巧與可程式化規則

- **兩手準備規則（IF/ELIF）**：
  ```
  IF 多頭贏 (站穩壓力位): THEN 找新高位壓制做空
  ELIF 跌破支撐確認: THEN 等回測再追空
  ELSE: 不進場，等明確訊號
  ```
- **嚴禁追漲殺跌**：行情已到支撐位（地板）時絕對不追空 → `IF (price - support) / support < threshold AND signal == 'SHORT': SKIP`
- **週末交易法則**：`IF is_weekend AND no_preset_entry: SKIP`，給點位才做，沒給就擺爛。
- **區間進場技巧**：壓力位往高處抓一點，給予區間（如 79,200 ~ 79,700）以應對小幅突破。
- **成本保護機制**：達到預期支撐位/獲利目標時，**先出部分倉位**保護利潤，剩餘倉位止損上移至開倉價（保本機制）。
- **遇支撐獲利了結**：下跌觸及強支撐時果斷平倉空單或至少出一半，等明確跌破才再進場。

## 一、BTC飛揚 — 短線網格與均線突破策略

**整體基調**：技術面右側交易，高頻或短線波段的趨勢跟蹤與均線回歸。

### 核心指標與觸發條件
- **均線系統 (MA/EMA)**：高度依賴 MA5、MA10、MA30 及 EMA7、EMA21、EMA55 排列。「死亡三角（死叉）」→ 做空訊號；「金三角」→ 多頭追漲訊號。
- **震盪指標 (RSI, KDJ, MACD)**：RSI 拐頭向下、KDJ 死叉向下、MACD 空頭縮量 → 進出場輔助濾網。
- **特殊訊號**：TD 序列（TD9、TD13 反轉訊號）判斷極限反轉點。
- **布林帶**：若價格貼著下軌且空間不足 → 機器人判定短線有反彈修復需求。

### 關鍵價位（2026-05-22 版本）
- **BTC 阻力區**：77,000 ~ 78,700 USD — 逢高做空 / 觸摸不過即做空
- **BTC 支撐**：回測均線支撐（MA30 / EMA55）
- **ETH 阻力**：2120 ~ 2160 USD；**支撐**：2000 ~ 2080 USD，跌破 2080 直接追空
- **山寨幣**：HYPE 回踩支撐 52 ~ 53 USD 低多；ZEC 看空 620 ~ 630 USD 接低

### 風控邏輯
- 嚴格等待「有效跌破」或「有效突破」才執行追單；壓力/支撐區間內震盪暫停發送交易訊號（避免假突破洗盤）

### 可程式化規則

### 關鍵價位（2026-05-22 版本）
